# Vue3 Canvas RPG Generator

## 技能概述

此技能用于生成基于 Vue3 + Canvas 的复古风格 RPG 游戏。游戏采用组件化架构，包含多区域地图探索、区域占领、回合制战斗（含小弟助战系统）、技能系统、装备系统、威望等级系统、天气系统、随机事件系统等核心玩法。

## 核心特性

### 1. 技术栈
- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **渲染**: Canvas 2D
- **存储**: localStorage
- **无第三方依赖**

### 2. 游戏系统

#### 多区域地图系统 ⭐ NEW
- **每个区域独立地图**: 30×25 瓦片（可扩展）
- **区域切换机制**: 通过出入口连接不同区域
- **动态加载**: 只渲染当前区域，优化性能
- **区域连接关系**: 树状或网状结构可配置
- **视觉提示**: 
  - 绿色出口 ➤ 可通行（目标区域已占领）
  - 红色出口 ➤ 不可通行（目标区域未占领）

#### 区域占领机制
- **4 个可占领区域**: 南市、闸北、虹口、法租界
- **BOSS 驻守**: 每个未占领区域有 BOSS 守护
- **完全占领**: 击败 BOSS 后自动标记整个区域
- **渐进式探索**: 必须按顺序占领才能进入新区域
- **UI 显示**: 已占领区域数/总区域数

#### 小弟助战系统 ⭐ NEW
- **战斗召唤**: 拥有小弟时自动参战
- **轮流攻击**: 每个小弟依次攻击敌人
- **动画效果**: 
  - 小弟 emoji 高亮显示
  - 攻击时前冲 + 放大 + 变红
  - 橙色伤害数字飘字
- **连击奖励**: 3 个以上小弟触发连击提示
- **伤害计算**: 基础伤害 + 人数加成 + 随机浮动

#### 技能系统 ⭐ NEW
- **5 个主动技能**:
  - ⚔️ 致命一击（冷却 5 回合，下次攻击×2，击败 50 敌人解锁）
  - 🛡️ 铁布衫（冷却 8 回合，防御 +50% 持续 3 回合，200 大洋解锁）
  - 💪 狂暴（冷却 10 回合，攻击 +50%/防御 -30% 持续 3 回合，威望 100 解锁）
  - 🩹 疗伤（冷却 6 回合，恢复 30HP，150 大洋解锁）
  - 🌪️ 小弟冲锋（冷却 8 回合，小弟伤害×1.5，10 个小弟解锁）
- **技能 UI**: 战斗界面显示技能按钮，冷却倒计时
- **自动解锁**: 满足条件时自动提示解锁
- **技能效果**: 实时影响战斗属性

#### 装备系统 ⭐ NEW
- **3 类装备**:
  - 🗡️ 武器（4 种：木棍/铁刀/钢剑/青龙刀）
  - 🛡️ 防具（4 种：布衣/皮甲/铁甲/金丝甲）
  - 💍 饰品（3 种：红宝石戒指/护身符/幸运符）
- **背包系统**: 购买后进入背包，可自由切换
- **装备界面**: 独立 UI 面板，显示当前装备和背包
- **属性加成**: 攻击/防御/威望加成实时计入
- **获取方式**: 酒馆购买/BOS S 掉落/威望兑换

#### 战斗系统
- **回合制战斗**: 弹窗显示，敌我双方属性
- **敌人分级**: 
  - Lv.1 杂鱼 (绿色) - 最弱
  - Lv.2 打手 (金色) - 中等
  - Lv.3 高手 (红色) - 最强
- **战斗选项**: 攻击 / 逃跑（BOSS 战不可逃跑）/ 使用技能
- **伤害公式**: `攻击力 - 防御力 + 随机浮动 (0-4)`（含装备/技能加成）
- **战斗奖励**: 金钱 + 威望，生命恢复 10%
- **失败惩罚**: 天数 -1，金钱减半，回到出生点

#### 酒馆商店
- **5 种商品**:
  - 🥤 跌打药酒（30 大洋，回复 30HP）
  - 💪 大力丸（80 大洋，永久攻击 +3）
  - 🛡️ 铁布衫（80 大洋，永久防御 +2）
  - 👥 招募小弟（50 大洋，永久攻击 +5，解锁助战）
  - ❤️ 医疗箱（100 大洋，满血复活）
- **实时刷新**: 购买后立即更新 UI 和属性
- **装备商店**: 独立装备界面，可购买武器/防具/饰品

#### 威望等级系统 ⭐ NEW
- **7 个威望等级**:
  - 街头混混（0 威望，初始）
  - 小喽啰（30 威望，攻击 +2/防御 +1）
  - 堂主（60 威望，攻击 +5/防御 +3）
  - 香主（100 威望，攻击 +10/防御 +5，解锁狂暴技能）
  - 护法（150 威望，攻击 +15/防御 +8，解锁金丝甲）
  - 上海滩霸主（200 威望，攻击 +20/防御 +10，胜利条件）
  - 传奇大亨（300 威望，攻击 +30/防御 +15，限定称号）
- **自动升级**: 威望达到阈值自动提升
- **属性加成**: 每级提供攻击/防御/小弟上限加成

#### 天气系统 ⭐ NEW
- **4 种天气**:
  - ☀️ 晴天（50%）：无影响
  - 🌧️ 雨天（25%）：移动 -20%/命中 -10%/隐蔽 +30%/防御 +10%
  - ❄️ 雪天（15%）：移动 -30%/命中 -15%/视野 -40%/隐蔽 +50%
  - 🌫️ 雾天（10%）：移动 -10%/命中 -20%/视野 -60%/隐蔽 +100%
- **变化规律**: 每 3 天自动变化
- **天气提示**: 变化时弹窗提示
- **UI 显示**: 右侧面板显示当前天气图标
- **战斗影响**: 雨天/雾天增加偷袭成功率

#### 随机事件系统 ⭐ NEW
- **奇遇事件**（每 20 步触发）:
  - 👨‍💼 神秘商人（5%）：稀有商品
  - 🤕 受伤武者（3%）：救助/离开/打劫
  - 💰 发现宝藏（2%）：100-300 大洋
  - ⚠️ 遭遇陷阱（3%）：HP-20/金钱 -30
  - ⚔️ 帮派火并（2%）：选择阵营，可能触发战斗
- **Buff 选择**（每占领 5 块地触发）:
  - ⚔️ 攻击强化（本局攻击 +20%）
  - 🛡️ 防御强化（本局防御 +20%）
  - ⚡ 神行符（本局移动 +30%）
  - 💰 招财符（本局金钱 +50%）
  - 👥 人和符（本局招募费用 -50%）
- **事件 UI**: 独立弹窗，多选项按钮
- **后果系统**: 奖励/惩罚/战斗触发

#### 时间系统
- **30 天倒计时**: 屏幕右侧显示
- **步数机制**: 每移动 10 步减少 1 天
- **工资系统**: 每天扣除小弟工资（小弟数×2 大洋）
- **金钱管理**: 可负债但不得低于 0

#### 胜负条件
- **胜利条件** (满足任一):
  - 占领所有区域（击败 4 个 BOSS）
  - 威望 ≥ 200
- **失败条件**: 
  - 天数归零且未达成胜利条件

#### 存档功能
- **完整状态保存**: 
  - 所有区域地图数据
  - 玩家属性（HP、攻击、防御）
  - 小弟数量、金钱、威望、天数
  - 区域占领状态
  - 敌人和 BOSS 位置
  - 玩家当前位置
- **一键读档**: 完全恢复游戏状态

    ├── GameCanvas.vue      # Canvas 渲染组件（地图、角色、特效）
    ├── UIPanel.vue         # 右侧信息面板（状态、存档、装备、天气按钮）

## 定制选项

### 可调整参数
1. **区域地图尺寸**: `REGION_MAP_WIDTH`, `REGION_MAP_HEIGHT` (默认 30×25)
2. **区域数量**: `REGIONS_TEMPLATE` 数组
3. **区域连接**: `exits` 配置（目标区域、出口坐标）
4. **敌人等级**: `level` 范围 (1-3)
5. **小弟数量**: `maxDisplay` (最多显示 5 个)
6. **商品价格**: `buyItem` 函数中的 price
7. **时间限制**: `stats.days` 初始值 (30 天)
8. **胜利条件**: 占领比例、威望阈值 (200)
9. **BOSS 属性**: `generateRegionBoss` 函数

## 代码质量要求

1. **响应式**: 使用 Vue3 的 `reactive`/`ref`/`computed`
2. **组件化**: 每个功能独立组件，职责单一
3. **状态管理**: `useGame` composable 统一管理游戏状态
4. **事件处理**: 使用 `$emit`/`@event` 进行父子组件通信
5. **条件渲染**: 使用 `v-if` 控制弹窗显示
6. **生命周期**: `onMounted`/`onUnmounted` 管理初始化和清理
7. **异步处理**: `async`/`await` 处理战斗回合延迟
8. **代码注释**: 关键函数和复杂逻辑必须有注释

## 注意事项

1. **资源绘制**: 所有元素使用 Canvas API 绘制，无需外部图片资源
2. **存档序列化**: 使用 `JSON.stringify` 和 `JSON.parse` 处理复杂对象
3. **战斗延迟**: 使用 `setTimeout` 和 `Promise` 实现回合动画延迟
4. **键盘监听**: 战斗/商店时临时移除，结束后恢复
5. **移动端适配**: 需添加虚拟方向键和触摸控制
6. **性能优化**: 
   - 只渲染可见区域（摄像机裁剪）
   - 使用 `requestAnimationFrame` 优化渲染
   - 大数据量时使用防抖/节流
7. **区域加载**: 切换区域时完整加载新地图，保存旧地图状态

## 核心函数说明

### useGame.js 主要函数

```javascript
// 初始化游戏
initGame()

// 生成所有区域地图
generateAllRegionMaps()

// 生成单个区域地图
generateSingleRegionMap(regionIndex)

// 加载指定区域
loadRegion(regionIndex)

// 玩家移动
movePlayer(dx, dy)

// 检查区域占领
checkRegionConquest()

// 生成区域 BOSS
generateRegionBoss(regionIndex)

// 战斗相关
attack()              // 玩家攻击（含小弟助战）
executeFollowerAttack()  // 执行小弟攻击
addDamageNumber(value, type)  // 添加伤害数字
flee()                // 逃跑
battleVictory()       // 战斗胜利
battleDefeat()        // 战斗失败

// 商店相关
buyItem(itemType)     // 购买商品

// 存档相关
saveGame()            // 保存游戏
loadGame()            // 读取游戏
```
