# 斧头帮之龙在江湖

> 一款复古风格的网页 RPG 游戏

## 🎮 游戏简介

你扮演斧头帮帮主阿龙，在 30 天内通过占地盘、战斗、招募小弟等方式扩张势力，最终称霸上海滩！

游戏采用纯 HTML5 Canvas 绘制，无需任何外部图片资源，复古黑金风格带你回到那个刀光剑影的年代。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

打开浏览器访问显示的地址即可开始游戏。

## 🎯 游戏目标

### 胜利条件（满足任一即可）
- 🏠 占领所有 4 个区域（击败所有 BOSS）
- ⭐ 威望达到 200 点

### 失败条件
- 📅 30 天内未能达成胜利条件

## 📖 玩法指南

### 一、基础操作

#### 移动控制
- ⌨️ **方向键** 或 **WASD** 控制上下左右移动
- 🗺️ 每次移动到一个新格子会自动占领该地盘
- 💰 占领奖励：大洋 +20，威望 +5

#### 界面说明
右侧面板显示你的当前状态：
- 👤 帮主：阿龙
- ⭐ 威望：当前威望值
- 💰 大洋：货币
- 👥 小弟：招募的小弟数量
- 🏠 地盘：已占领区域数/总区域数
- 📅 剩余天数：距离游戏结束的天数
- ☀️ 天气：当前天气状况

### 二、区域探索

#### 区域划分
游戏分为 4 个区域：
1. **南市** - 起始区域（默认已占领）
2. **闸北** - 第一个挑战区域
3. **虹口** - 高级区域
4. **法租界** - 最终区域

#### 出入口机制
- 🟢 **绿色出口**：可通行（目标区域已占领）
- 🔴 **红色出口**：不可通行（目标区域未占领）
- 📍 必须击败区域 BOSS 才能占领该区域

### 三、战斗系统

#### 遭遇敌人
- 👹 地图上随机分布 30 个敌人出没点
- 🎯 走到敌人点上自动触发战斗
- ⚔️ BOSS 驻守在各区域关键位置

#### 敌人分级
| 等级 | 名称 | 颜色 | 强度 |
|------|------|------|------|
| Lv.1 | 杂鱼 | 🟢 绿色 | 最弱 |
| Lv.2 | 打手 | 🟡 金色 | 中等 |
| Lv.3 | 高手 | 🔴 红色 | 最强 |
| 👹 | BOSS | 👹 恶魔 | 极强 |

#### 战斗选项
1. **⚔️ 攻击** - 普通攻击
   - 双方轮流攻击
   - 伤害 = 攻击力 - 防御力 + 随机浮动 (0-4)
   
2. **🏃 逃跑** - 退出战斗
   - 消耗：威望 -5
   - BOSS 战不可逃跑

3. **✨ 技能** - 使用主动技能
   - 见下方技能系统详解

#### 战斗结果
- **胜利** 💪
  - 获得金钱和威望
  - 生命恢复 10%
  - 敌人据点消失

- **失败** 😵
  - 天数 -1
  - 金钱减半
  - 回到出生点
  - 生命回满

### 四、技能系统

#### 主动技能（5 个）

| 技能 | 图标 | 冷却 | 效果 | 解锁条件 |
|------|------|------|------|----------|
| **致命一击** | ⚔️ | 5 回合 | 下次攻击伤害×2 | 击败 50 个敌人 |
| **铁布衫** | 🛡️ | 8 回合 | 防御 +50%，持续 3 回合 | 花费 200 大洋 |
| **狂暴** | 💪 | 10 回合 | 攻击 +50%，防御 -30%，持续 3 回合 | 威望≥100 |
| **疗伤** | 🩹 | 6 回合 | 恢复 30HP | 花费 150 大洋 |
| **小弟冲锋** | 🌪️ | 8 回合 | 小弟伤害×1.5 | 拥有 10 个小弟 |

#### 技能使用
- 战斗界面中部显示技能按钮
- 灰色 = 未解锁
- 金色 = 可用
- 橙色数字 = 冷却中回合数
- 红色 = 持续性技能激活中

### 七、天气系统

#### 天气类型
| 天气 | 图标 | 概率 | 效果 |
|------|------|------|------|
| **晴天** | ☀️ | 50% | 无影响 |
| **雨天** | 🌧️ | 25% | 移动 -20%，命中 -10%，隐蔽 +30%，防御 +10% |
| **雪天** | ❄️ | 15% | 移动 -30%，命中 -15%，视野 -40%，隐蔽 +50% |
| **雾天** | 🌫️ | 10% | 移动 -10%，命中 -20%，视野 -60%，隐蔽 +100% |

#### 天气变化
- 🔄 每 3 天自动变化
- 📢 变化时弹窗提示
- 📊 右侧面板显示当前天气

#### 天气策略
- 🌧️ **雨天/雾天**：适合偷袭，隐蔽性大幅提升
- ❄️ **雪天**：适合防守，敌人移动困难
- ☀️ **晴天**：正常作战

### 八、随机事件系统

#### 奇遇事件（每移动 20 步触发）

1. **👨‍💼 神秘商人** (5% 概率)
   - 看看商品 → 打开商店
   - 离开 → 无事发生

2. **🤕 受伤武者** (3% 概率)
   - 救助（花费 50 大洋）→ 随机奖励（秘籍/威望/金钱）
   - 离开 → 威望 -5
   - 打劫 → 获得 100 大洋，威望 -20，可能触发战斗

3. **💰 发现宝藏** (2% 概率)
   - 拿走 → 获得 100-300 大洋

4. **⚠️ 遭遇陷阱** (3% 概率)
   - 挣脱 → HP -20，金钱 -30

5. **⚔️ 帮派火并** (2% 概率)
   - 帮助斧头帮 → 胜利：金钱 +200/威望 +30；失败：HP -50
   - 帮助青龙帮 → 胜利：威望 +20/获得装备；失败：HP -50
   - 跑路 → 安全离开

#### Buff 选择（每占领 5 块地触发）

选择一项临时增益（持续到游戏结束）：
1. ⚔️ **攻击强化** - 攻击力 +20%
2. 🛡️ **防御强化** - 防御力 +20%
3. ⚡ **神行符** - 移动力 +30%
4. 💰 **招财符** - 金钱获取 +50%
5. 👥 **人和符** - 招募费用 -50%

### 十、小弟系统

#### 招募小弟
- 🍶 在酒馆花费 50 大洋招募
- 💪 每个小弟永久增加攻击力 +5
- ⚔️ 战斗中自动助战

#### 小弟助战
- 🎯 拥有小弟时，战斗中自动参战
- ⚔️ 每个小弟轮流攻击一次
- 💥 显示独立伤害数字（橙色）
- 🎊 3 个以上小弟触发"连击"提示

#### 小弟伤害计算
```
单个小弟伤害 = 3 + min(小弟数量，10) × 0.5 + 随机 (0-2)
总伤害 = 所有小弟伤害之和
```

### 十一、时间系统

#### 时间流逝
- 📅 初始 30 天
- 🚶 每移动 10 步减少 1 天
- 💰 每天扣除工资 = 小弟数量 × 2 大洋

#### 金钱管理
- 💵 可负债（金钱变为负数）
- ⚠️ 但实际支付不得低于 0
- 📉 合理安排小弟数量，避免破产

## 💡 游戏技巧

### 新手建议

1. **前期发展** 🌱
   - 优先占领南市（起始区域）
   - 尽快招募 2-3 个小弟
   - 购买铁刀和皮甲提升属性

2. **中期扩张** 📈
   - 击败闸北 BOSS，占领第二个区域
   - 威望达到 100 解锁狂暴技能
   - 保持金钱储备，避免负债过多

3. **后期冲刺** 🏃
   - 合理选择 Buff（推荐攻击/防御强化）
   - 利用雨天/雾天偷袭 BOSS
   - 预留足够天数挑战最终 BOSS

### 战斗策略

1. **技能使用** ✨
   - 致命一击：用于斩杀残血敌人
   - 铁布衫：对抗高攻击敌人
   - 狂暴：速战速决时使用
   - 疗伤：危急时刻保命
   - 小弟冲锋：小弟数量多时使用

2. **天气利用** 🌤️
   - 雨天/雾天：隐蔽性高，适合偷袭
   - 雪天：敌人移动困难，适合防守
   - 晴天：正常作战

3. **装备搭配** ⚔️
   - 前期：铁刀 + 皮甲（性价比高）
   - 中期：钢剑 + 铁甲（属性均衡）
   - 后期：青龙刀 + 金丝甲（顶级装备）

### 威望速刷

1. 🏠 多占地盘（+5/块）
2. ⚔️ 多击败高等级敌人（+30/个）
3. 🎖️ 完成随机事件任务
4. ⚠️ 避免逃跑（-5/次）

## 💾 存档功能

### 保存游戏
- 点击右侧面板"保存游戏"按钮
- 自动保存到浏览器 localStorage
- 保存所有状态（位置、属性、装备、地图等）

### 读取游戏
- 点击右侧面板"读取游戏"按钮
- 完全恢复上次保存的状态
- 可随时继续游戏
